Super cool vote plugin for wordpress.

== Description ==

Voting made easy to display thumbsup style vote under any posttype. easy to configure and use various themes.



<strong>Plugin Features</strong>

* Different themes.
* Post-type selection.
* Responsive design.
* Light weight.
* Unlimited or single voting.






== Installation ==


1. Install as regular WordPress plugin.
2. Go your Pluings setting via WordPress Dashboard and <strong>activate</strong> it.
3. After Activate you see Menu on left Sidebar "<strong>Voting</strong>" 

if you are first time change settings and hit <strong>"Save Changes"</strong>

and visit your website or single post.
